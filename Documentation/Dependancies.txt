Dependancies:

The most current version of Ubuntu is reccommended,
however any Unix-like OS with the required tools will probably work

What to install:
1) i686-elf 4.9.1 cross compiler toolchain
	A) Obtaining it:
		Can be downloaded here: http://newos.org/toolchains/i686-elf-4.9.1-Linux-x86_64.tar.xz
		Or build from these instructions: http://wiki.osdev.org/GCC_Cross-Compiler
	B) Installation
	Make sure that the toolchain is in "~/opt/cross"

2) nasm assembler
	A) Obtaining it:
		for Ubuntu or debian, run "sudo apt-get install nasm"
